About StegVault

What is StegVault?

StegVault is a Universal Steganography Toolkit that allows you to hide encrypted secrets (text or files) inside ordinary images, videos, audio, and documents.

How it Works

  • Image Steganography: Uses Least Significant Bit (LSB) embedding. It slightly modifies the lowest bits of the pixels in PNG/WebP images to store data. Visually, the change is imperceptible.
  • Video, Audio & Document Steganography: Uses End-Of-File (EOF) injection. Since video and audio encoders in browsers are lossy, traditional LSB would be destroyed. Instead, the encrypted payload is safely appended to the end of the file along with a special signature. This works universally across formats like MP4, MP3, PDF, and ZIP. This means unlimited capacity and zero quality degradation!
  • Encryption: All data is compressed first, then encrypted using military-grade AES-256-GCM cryptography before being embedded. Your password is never stored anywhere.

Capabilities & Uses

  • 100% Local: Everything runs locally in your browser. No data is ever sent to any server.
  • Secure Communication: Send secret messages over public channels where the carrier file appears completely innocuous.
  • Data Archival: Store private backups inside large video, audio, or document files without arousing suspicion.

Limitations & Warnings

  • If you upload the generated stego file to a social media platform that recompresses or strips metadata (like WhatsApp or Twitter), the hidden data will be destroyed. For safe transmission, send files as "Documents" or via uncompressed channels like email, USB, or cloud storage.
  • Image capacity is strictly limited by the number of pixels. If you try to hide too much data in a small image, it will fail (or look noisy).

Future Enhancements

  • More advanced LSB techniques (like randomized bit dispersion).
  • Dark/Light mode themes.
